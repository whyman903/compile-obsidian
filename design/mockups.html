MyWiki — redesign proposals

Three directions + the current state for comparison. Popover is 460pt wide in all cases.

Current

As-is

Rainbow gradients, glowing tiles, sparkles iconography. Opacity used for hierarchy. Reads as a 2023 AI-wrapper template.

Option A

Editorial

Warm near-black, New York serif for display, single brass accent. Inspired by iA Writer and Things 3. Reads as bookish and calm.

Option B

Graphite

Cold charcoal, JetBrains Mono for metadata, single mint accent. Raycast / Linear / Warp language. Reads as a precise power tool.

Option C

Native Mac

Defers to macOS — system materials, system accent color, semantic typography. Lowest risk, most forgettable.

These are wireframes with a color mood, not pixel comps. They don't simulate real macOS vibrancy/blur, shadow depth, hover states, or animation. Judge them on layout, palette, and typographic voice. Full writeup + rationale in ui-proposal.md.
